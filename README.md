# 🚀 Portfolio - Joseph HOMAWOO

Bienvenue sur le dépôt de mon portfolio personnel. Ce site présente mes projets, mes compétences en développement (Back-end, Mobile, Web) et mon expérience en tant que Designer UX/UI.

## 🛠 Stack Technique

Ce portfolio est construit avec des technologies statiques pour garantir performance et sécurité, hébergé via **GitHub Pages**.

*   **SSG :** [Jekyll](https://jekyllrb.com/) (Ruby)
*   **Moteur de Templates :** Liquid
*   **Style :** Sass (SCSS) & Bootstrap
*   **Scripts :** jQuery & JavaScript (Plugins : Slick, Shuffle)
*   **Déploiement :** GitHub Actions

---

## 🌟 Fonctionnalités

- **Section Hero :** Présentation rapide.

## 💻 Installation Locale

Pour exécuter ce projet sur votre machine, assurez-vous d'avoir [Ruby](https://www.ruby-lang.org/) et [Bundler](https://bundler.io/) installés.

1.  **Cloner le dépôt :**
    ```bash
    git clone https://github.com/josephhomawoo/josephhomawoo.github.io.git
    cd josephhomawoo.github.io
    ```

2.  **Installer les dépendances :**
    ```bash
    bundle install
    ```

3.  **Lancer le serveur de développement :**
    ```bash
    bundle exec jekyll serve
    ```
    Accédez ensuite à `http://localhost:4000` dans votre navigateur.

## 📂 Structure du Projet

- `_data/` : Contient les fichiers YAML (`settings.yml`, `plugins.yml`) pour gérer le contenu sans toucher au code.
- `_includes/` : Composants réutilisables (header, footer, sections).
- `_layouts/` : Gabarits de pages (default, post, page).
- `_pages/` : Contenu des pages principales (About, Portfolio, Contact, etc.).
- `assets/` : Ressources statiques (images, scss, js, plugins).
